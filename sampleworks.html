A Collection of My Projects

MapView for UWTicketBuddy

UWTicketBuddy is a project started at HackGT by me, Brian Glowniak, Daniel Hathcock, and Yu-Lin Yang. It is an app that is meant to streamline the process of buying and selling football tickets at the University of Wisconsin - Madison because football is such a big deal there. I worked on the implementation of the Google Maps API and was able to get location services to work, which will help buyers and sellers see who is near them.

You can view the code on GitHub here.
