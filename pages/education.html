Classes Taken

CS 1331: Intro to Object Oriented Programming

In this class, I learned fundamental Java programming skills that are an essential base for higher level classes. We also covered basic object oriented programming principles such as inheritance and polymorphism, abstraction, and encapsulation. The project I worked on from CS 1331 can be found here.

CS 1332: Data Structures and Algorithms

In CS 1332, we utilized Java to cover essential data structures and algorithms in computer science. After taking this class, I am familiar with common data structures like stacks, queues, maps, and binary search trees. I am also familiar with graph and string searching algorithms. Like CS 1331, this class helped me establish a solid foundation of skills for the future.

CS 2051: Honors Discrete Mathematics

CS 2051 has helped me think more creatively about mathematics and computer science through learning about topics such as graph theory, cryptography, counting, and boolean logic. Through CS 2051, I have a better understanding of how computers process and perform calculations, it has definitely helped me improve as a coder.

Current Classes (Fall 2017)

CS 2110: Intro to Computer Organization and Programming

CS 2110 is all about the most fundamental workings of a computer. We will be learning about logic gates, ALUs, how a memory is made, and C programming throughout this Fall semester. This is a fun class and I'm excited to learn exactly how a computer works.

CS 2340: Objects and Design

As a CS student, I am highly interested in entering the field of software development in the future. CS 2340 provides a perfect environment for me to experience what being a software developer is like. Throughout the course of this semester, we will be learning about various developement models such as the Agile Developement model, and we will be polishing our knowledge in object oriented programming. I am currently part of a team of 5 and we will be creating an app in Android Studio this Fall.

ISYE 3770: Statistics and Applications

I'm excited to learn about probability and statistics because I believe that it will help me develop important analysis skills that will stay with me in anything that I do as I progress through my career. We will be learning about discrete and continuous probability as well as various analytical methods for statistics.
